测试分享

　　Vue.js（读音 /vjuː/, 类似于 view）是一个构建数据驱动的 web 界面的库。Vue.
js的目标是通过尽可能简单的API 实现响应的数据绑定和组合的视图组件。

　　Vue.js 自身不是一个全能框架——它只聚焦于视图层。因此它非常容易学习，非常容易与其它库或已
有项目整合。另一方面，在与相关工具和支持库一起使用时，Vue.js也能完美地驱动复杂的单页应用。

　　最近折腾了一个礼拜，熟悉vue，在这里简单介绍一下安装配置。
首先要确保你已经安装了node.js，然后打开命令行，输入如下命令进行安装：

$ npm install -g vue-cli
// 创建一个基于 "webpack" 模板的新项目
$ vue init webpack my-project
// 进入项目目录，安装依赖
$ cd my-project
$ npm install
//启动项目
$ npm run dev

　　启动成功命令行会出现如下提示：

　　浏览器输入localhost:8080出现如下所示，即为安装成功。

　　如果端口冲突，可以手动改变监听端口，项目->build->dev-server.js,将底部8080更改为其他即可，保存并命令行重启既能以新端口访问。


　　安装成功了，看看如何使用吧，点此前往vue文档。

app.vue

<script>
import Hello from './components/Hello'

  export default {
    components: {
      Hello
  }
}
</script>

　　引入了hello组件，hello组件里的内容会插入到标签内。
hello.vue

<script>
    export default {
      data () {
        return {
          msg: 'Hello ~~ ~'
        }
      }
    }
</script>

　　保存之后浏览器访问即可见到变化哦，很神奇的呢~~~

　　msg里的内容进行界面渲染，就被Hello ~~~进行填充了。是不是很简单？啦啦啦！！
